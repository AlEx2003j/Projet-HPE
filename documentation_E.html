The structure

To realize this project we used several technologies. First we used HTML and CSS to build the structure of the website. The internal structure also includes the tree structure of the website, that is how the pages communicate with each other.

In a second time we used javascript to give some effects to the site. This is the case of the appearance animations which are entirely made in javascript.

The javascript was also used to realize the effects of having for the games. Indeed to realize this part the Hammer library was used. This library was specially designed for projects requiring animations similar to those of Tinder.

The contact form

The visible part was realized in HTML and CSS, but for the data to be sent it was necessary to use the services of Formspree. This service could be integrated in a javascript file with a function that is previously given. In fact, before the mail arrives on the client's mailbox, it first goes through an internal mailbox of the service.

The translation

As far as the translation of the site is concerned, it is not done by using an external api (application programming interface). In fact, the site has been developed twice identically with the only parameter changing the language. There is therefore a second version of the website in English. We wanted to develop the site in French because it is our original language, but also in English so that some passages of our work that are quoted or reformulated can be in the language in which we studied them.
