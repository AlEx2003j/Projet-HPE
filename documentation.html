La structure

Pour réaliser ce projet nous avons utilisé plusieurs technologies. Dans un premier temps nous avons utilisé le HTML et le CSS pour construire la structure même du site. La structure interne comprend aussi l’arborescence du site web, c'est-à-dire comment le page communique entre elles.

Pour réaliser ce projet nous avons utilisé plusieurs technologies. Dans un premier temps nous avons utilisé le HTML et le CSS pour construire la structure même du site. La structure interne comprend aussi l’arborescence du site web, c'est-à-dire comment le page communique entre elles.

Le javascript a aussi été utilisé pour réaliser les effets de swipe pour le jeu. En effet pour réaliser cette partie la bibliothèque Hammer a été utilisée. Cette bibliothèque a été spécialement conçue pour les projets nécessitant des animations semblables à celles de Tinder.

Le formulaire de contact

La partie visible a été réalisée en HTML et CSS, mais pour que les données soient envoyées, il a fallu utiliser les services de Formspree. Ce service a pu s’intégrer dans un fichier javascript avec une fonction qui est donnée au préalable. Dans les faits, avant que le mail arrive sur la boîte mail du client, il passe d’abord par une boîte mail interne au service.

La traduction

En ce qui concerne la traduction du site elle n’est pas faite par l’utilisation d'un API (application programming interface) externe. En réalité, le site a été développé deux fois à l’identique avec comme seul paramètre changeant la langue. Il existe donc une deuxième version du site internet en anglais. Nous avons voulu développer le site en français car c’est notre langue d’origine, mais également en anglais pour que certains passages de notre travail qui sont cités ou reformulés puissent être dans la langue dans laquelle nous les avons étudiés.
